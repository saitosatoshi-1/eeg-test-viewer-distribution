Windows 共同研究者への配布用フォルダです。

共同研究者へ渡すファイル:

  配布用_Windows_EEG_Viewer.zip

共同研究者側の手順:

1. ZIP を解凍する
2. 解凍された EEG Viewer フォルダを開く
3. Setup EEG Viewer Windows.bat をダブルクリックする
4. セットアップ完了後、Desktop または Start Menu の EEG Viewer アイコンから起動する

GitHub へのアクセス、GitHub の ID/PW 入力は不要です。

注意:
- Windows に Miniforge / Miniconda / Anaconda のいずれかが必要です。
- 入っていない場合、セットアップ画面で Miniforge のインストール先が案内されます。
- 配布前に ZIP を更新したい場合は、このフォルダ内の ZIP を作り直してください。
